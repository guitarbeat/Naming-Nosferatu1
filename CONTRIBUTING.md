# Contributing to Name Nosferatu

Thank you for contributing! For our full, detailed guidelines, please read the comprehensive **[Contributing Guide](docs/CONTRIBUTING.md)**.

## Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/guitarbeat/name-nosferatu.git
   cd name-nosferatu
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Run tests**
   ```bash
   pnpm test
   ```

4. **Submit a Pull Request**
   - Create a feature branch off `main`
   - Run `pnpm run check` locally to ensure code passes all checks
   - Ensure your PR title follows Conventional Commits format (e.g., `feat: add new button`)
   - Push and open a PR!
